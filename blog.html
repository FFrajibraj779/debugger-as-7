2.What are difference between global scope and block scope js ?

Answer: the variables defined outside of any function or curly brackets are known as global variables and have global scope. global scope means that the variables can be accessed from any part of that program, any fucntion or conditional state can access that variable. a block scoped variable means that the variable defined within a block will not be accessible form outside the block.let, const is a block scope and var is a global scope.
